05 — Usage Guidelines

Dos and Don'ts

Protect the brand by following these guidelines in every context.

✓ Clear Space

Maintain minimum clear space equal to the height of the inner "a" on all sides of the mark.

✓ Minimum Size

Never render the icon smaller than 24x24px. Use the primary indigo version for most contexts.

✓ Approved Variants

Use only the provided color variants. Indigo is primary, cyan-accent for marketing, mono for print/legal.

✓ Background Contrast

Use color variants on white/light. Use reversed (white) on dark backgrounds. Use navy bg for app icons.

✗ Stretch or Skew

Never distort the proportions. Always scale uniformly. The circle must remain a perfect circle.

✗ Add Effects

No drop shadows, outlines, bevels, or glows beyond what's defined in the provided assets.

✗ Recolor

Don't apply arbitrary colors. Only use the provided variants — indigo, cyan, purple, white, black.

✗ Separate Elements

Don't use the "a" without its ring, or detach the "a" connection point from the outer circle.
